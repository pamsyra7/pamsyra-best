# SH건축사사무소 (SH Architecture)

> 제주의 청정자연을 담은 에코프렌들리 건축사사무소

---

## 소개

SH건축사사무소는 **제주 여성건축사** 가 이끄는 제주 기반의 건축사사무소입니다.

건축 설계·감리를 핵심으로, 전문 설계 컨설팅을 제공합니다.
에너지 효율, 장애인 편의, 교육시설 인증 등 인증·평가 서비스까지 통합적으로 지원합니다.

---

## 주요 서비스

### 🏗️ 설계 · 감리
- 건축 설계 및 공사감리
- 건축물 해체감리
- 석면해체감리

### 🌿 치유농업 전문 컨설팅
- 치유농장 설계 전문 컨설팅
- 치유농업 공간 프로그램 기획

### 📋 인증 · 평가 서비스
- 건축물 에너지효율 평가
- 장애인 편의시설(VF) 설계 및 인증
- 학교교육 인증시설 심사 *(1급 등급심사위원)*
